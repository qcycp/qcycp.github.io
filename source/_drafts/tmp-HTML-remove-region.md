取消網頁預留邊界
===
http://earthcat99.pixnet.net/blog/post/36811165-html%E5%8F%96%E6%B6%88%E9%A0%90%E7%95%99%E9%82%8A%E7%95%8C%E7%9A%84%E6%96%B9%E6%B3%95

讓網頁內容從左上方 0,0 座標的位置開始
設定左方邊界與上方邊界為 0 即可

```
<!--
topmargin="0" 設定上邊界
leftmargin="0" 設定左邊界
-->
<body topmargin="0" leftmargin="0">
```
