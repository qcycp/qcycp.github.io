Hello World for React

發表於 2020-06-04 更新於 2020-07-09 分類於 Programming ， React
  1. install react installation package
    $ sudo npm install -g create-react-app
  2. create react app
    $ create-react-app hello-world
  3. start react server on port 3000
    $ cd hello-world
    $ npm start
  4. install json server
    $ npm install -g json-server
  5. start json server
    到當前的 project folder
    --watch可以監控當 data 有修改時，會自動 reload server
    $ json-server.cmd --watch --port 3004 ./src/db.json
